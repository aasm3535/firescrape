Documentation

FireScrape is a dual-purpose tool: a high-speed web scraping API and a Model Context Protocol (MCP) server for AI agents.

Installation

NPM Package (MCP Server)

bun add @yutugyutugyutug/firescrape-mcp

Scrape API

GET request to extract content, metadata, and bypass bot protections.

GET /api/scrape
url The target URL to scrape (required)

Search API

Anonymous web search powered by DuckDuckGo.

GET /api/search
q Search query (required)

MCP Server

Connect FireScrape to Claude Desktop or Cursor to give your AI web access.

Quick Run

Run directly without installation (using Bun or Node):

bunx @yutugyutugyutug/firescrape-mcp

Claude Configuration 

"firescrape": {
  "command": "bunx",
  "args": ["@yutugyutugyutug/firescrape-mcp"]
}
