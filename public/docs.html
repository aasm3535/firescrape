Introduction

FireScrape is a simple, powerful API for extracting data from the web. It handles the hard parts: rotation, headers, and parsing.

Scrape

GET /api/scrape?url=...

Returns the content of a page, including metadata and hidden JSON structures.

Search

GET /api/search?q=...

Privacy-focused web search powered by DuckDuckGo.
