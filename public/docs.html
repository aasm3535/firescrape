Introduction

FireScrape is a serverless-first web scraping API designed for modern frontend developers. It handles headless browsers, user-agent rotation, and CAPTCHA detection under the hood.

Base URL https://firescrape-plum.vercel.app

Authentication

The API is currently open for public use in beta. No API key is required.

Scrape Endpoint

Extract metadata, main content, and hidden structured data from any public URL.

GET /api/scrape
Param Type Description
url string The target URL to scrape. Must be URL-encoded.

Response Object


{
  "success": true,
  "data": {
    "url": "https://example.com",
    "meta": { ... },
    "content_preview": "..."
  }
}

MCP Server

FireScrape provides an official Model Context Protocol (MCP) server. This allows AI assistants like Claude or Cursor to search the web and scrape pages using our infrastructure directly.

firescrape-mcp

Get the official MCP server code, installation guide, and Claude Desktop configuration.
